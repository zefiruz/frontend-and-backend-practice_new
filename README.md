# Портфолио студента

Веб-сайт портфолио в стиле 2000-х годов.

## Структура проекта

- `index.html` - главная страница
- `pages/` - дополнительные страницы
- `styles/` - CSS стили
- `images/` - изображения
- `assets/` - дополнительные файлы

## Технологии

- HTML5
- CSS3
- JavaScript
